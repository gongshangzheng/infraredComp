Notes

  • FLIR ADAS 1.3 images are 14-bit thermal data stored as 16-bit TIFF (640×512).
  • JPEG/WebP require 8-bit input — images are normalized via min-max scaling before encoding.
  • PNG, JPEG2000, and WebP-lossless are lossless — PSNR is ∞ (perfect reconstruction).
  • Learned codecs use CompressAI pretrained models; input is padded to multiples of 64.
  • Timing measurements are single-threaded on CPU (Apple arm).
