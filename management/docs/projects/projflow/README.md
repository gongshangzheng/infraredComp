# ProjFlow

轻量级项目管理平台，适用于小型团队的日常协作管理。

## 项目简介

ProjFlow 是一个开箱即用的项目管理 demo 平台，基于 Markdown 文件和 SQLite 存储数据，
提供团队管理、日报/周报/月报、任务看板、里程碑、会议纪要等协作能力。
本仓库同时是项目树功能的示例项目——你看到的这棵任务树，就是 ProjFlow 自身的迭代计划。

## 技术栈

- **前端**：Vue 3 + Vite + Naive UI + Pinia
- **后端**：FastAPI (Python)
- **数据源**：Markdown 文件 + SQLite

## 功能模块

1. **项目管理**：团队、报表、任务看板、里程碑、会议纪要、项目树
2. **论文搜集**：论文导入、分类、笔记、搜索筛选
3. **评测体系**：模型管理、数据集管理、评测运行、结果对比

## 协作方式

大部分操作可通过 AI 协作完成，配套 Claude Code skills 提供操作指南。
